# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added

- Comprehensive test suite with Jest
- TypeScript definition file (.d.ts)
- JSDoc documentation for all public APIs
- Input validation and error handling
- ESLint for code quality
- Prettier for code formatting
- GitHub Actions CI/CD pipeline
- CodeQL security scanning
- Pre-commit hooks with Husky and lint-staged
- Extensive documentation (CONTRIBUTING.md, CODE_OF_CONDUCT.md, SECURITY.md)
- Issue and PR templates
- Examples directory with usage examples
- Performance benchmarks
- Browser compatibility testing

### Changed

- Updated mathjs to v13.2.2 (from v6.0.0)
- Enhanced error messages with context
- Improved README with comprehensive examples

### Fixed

- Package.json syntax error (missing comma)

## [0.1.0] - 2016-XX-XX

### Added

- Initial release
- Basic mathematical expression evaluation using mathjs
- Support for arithmetic, trigonometry, complex numbers, matrices, and units
